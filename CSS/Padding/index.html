Padding

CSS tem propriedades para especificar o preenchimento de cada lado de um elemento

Esse elemento div possui paddin-top: 50px; padding-right: 30px; padding-bottom: 50px; Padding-left:80px;

Preenchimento - Propriedade abreviada

Para encurtar o código, é possivel especificar todas as propriedades de preenchimento em uma propriedade

paddin-top: 25px;
padding-right: 50px;
padding-bottom: 75px;
Padding-left: 100px;
